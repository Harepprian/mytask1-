TEXT PROPERTIES
Property Values Description
text-decoration none, underline, overline, line-through Adds decoration lines to text
text-decoration color color, inherit Sets the color of the text decoration.
text-decoration line none, underline, overline, line-through Specifies which decoration line to apply
text-decoration style solid, dashed, dotted, double Defines the style of the text decoration line
text-transform uppercase, lowercase, capitalize, none controls the capitalization of the text
text-shadow offset-x offset-y blur-radius color Applies shadow to the text
text-align left, right, center, justify Aligns text horizantally
text orientation upright, sideways Specifies the orientation of text in vertical writing modes
text-justify auto, inter-word, inter-character Specifies the justification method used for text alignment
text-opacity number (0 to 1) . Controls the opacity level of the text
FONT PROPERTIES
Property Values Description
font-family Arial, Georgia, sans-serif Specifies the font family for the text.
font-size 16px, 1em, larger Defines the size of the font.
font-weight normal, bold, 100, 400 Sets the thickness of the font characters.
font-style normal, italic, oblique Defines the style of the font (normal, italic, or oblique).
font-variant normal, small-caps Specifies whether to use small caps or not.
line-height 1.5, 150%, 1.2em Sets the amount of space between lines of text.
text-indent 20px, 2em, 5% Indents the first line of a block of text.
letter-spacing normal, 2px, 0.1em Adjusts the spacing between characters.
word-spacing normal, 4px, 0.2em Adjusts the spacing between words.
font-variant-caps normal, small-caps, all-small-caps Controls the capitalization style of the font.
